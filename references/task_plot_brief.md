# Task Plot Brief

## Task

- Title: Sternberg Working Memory Task
- Construct: working memory / memory scanning / load manipulation
- Paradigm: item-recognition task with memory-set size manipulation.

## Rows

- Set size 3: memorize 3 letters.
- Set size 5: memorize 5 letters.
- Set size 7: memorize 7 letters.

## Trial Timeline

1. Memory set: 1500 ms. Show letter set for current load, no response.
2. Retention: 2500 ms fixation delay, no response.
3. Probe response: 3000 ms. Show probe letter. Press F for OLD, J for NEW.
4. Feedback: 800 ms. Show correct, incorrect, or timeout feedback and total score.
5. ITI: 1000 ms fixation-only interval, no response.

## Notes

- Probe-old probability is 0.5.
- Letter pool is B, D, F, G, H, K, L, M, P, Q, R, T.
- Correct feedback gives +1 point; incorrect and timeout give 0.
